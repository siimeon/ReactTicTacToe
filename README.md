# Tic Tac Toe

## Requirements

* Python 2 & Python 3
* Chrome or Firefox

## How to run

```
./start.sh
```

or

```
python -m SimpleHTTPServer 8000  # Python 2
python -m http.server --cgi 8000 # Python 3
```
